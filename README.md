# YouTube Pause Chrome Extension

## Overview
The YouTube Pause Chrome Extension allows users to automatically pause YouTube videos when they switch tabs or minimize the browser, providing a seamless user experience.

## Features
- Automatically pauses videos when the tab is inactive
- Resumes playback when returning to the tab
- Customizable settings for user preferences

## Installation
1. Download the extension files from the repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode".
4. Click on "Load unpacked" and select the downloaded extension folder.

## Usage
- Once installed, the extension runs in the background and requires no additional setup. Just navigate to YouTube and start watching videos.

## Configuration
- Users can access the options page by clicking the extension icon in the toolbar. Here, settings such as enabling/disabling the auto-pause feature can be adjusted.
